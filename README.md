# Tetris

## Dependencies
- `ncurses` for the terminal interface
- `make` and `gcc` with c++17 support for compiling

## Building
```bash
cd src
make
```

## Running
```bash
./tetris
```

## Controls
- `q`: quit the game
- `left`: move left
- `right`: move right
- `down`: move down (soft drop)
- `y`: rotate left
- `x`: rotate right
